Building Pangool

Getting Pangool

You can download Pangool from the download page or you can clone the latest version from Github: 

git clone git://github.com/datasalt/pangool.git

Building Pangool

Apache Maven is needed to build Pangool. You can build Pangool by launching the following command on Pangool home:X 

mvn install
