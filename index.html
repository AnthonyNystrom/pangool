Pangool

Hadoop has a steep learning curve. Pangool has the goal to simplify Hadoop development without loosing the performance and flexibility that the low level Hadoop API provides.

The most common patterns that arises when writing MapReduce jobs are easier to implement with Pangool with similar performance. That is thanfully to implement the new paradigm Tuple MapReduce

Easy secondary sorting

Although it is a very common pattern, secondary sorting is a nightmare with Hadoop.

Check it out how easy it is with Pangool. 

job.setGroupByFields("word");
job.setOrderBy(new OrderBy()
  .add("word", Order.ASC)
  .add("count", Order.DESC));
